This starter is a working docs site. Clone it, connect it to Bloques, and replace the content with your own. Every page demonstrates a feature you can use in your own pages — the markdown you read here is the same markdown that produced it.

## What's inside

- `docs.json` — site name, navigation, theme, and navbar.
- `index.mdx` — the homepage you're reading now.
- `quickstart.mdx` — onboarding for the reader of your docs.
- `essentials/` — writing basics and component examples.
- `configuration/` — navigation and theme reference.

Start with the [Quickstart](/quickstart) to set up the repo, then replace each page with your own content.
